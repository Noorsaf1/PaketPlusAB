Ladda ner och packa upp filen på ditt lokala system och kopiera mappen "carrental".
Placera mappen "carrental" i rotkatalogen (för xampp i xampp/htdocs, för wamp i wamp/www, för lamp i var/www/html).
Databaskonfiguration:

Öppna phpmyadmin.
Skapa en databas med namnet carrental_db.
Importera databasen carrental_db.sql (tillgänglig i mappen SQL File Folder inuti zip-paketet).



För adminpanel:

Öppna din webbläsare och skriv in “http://localhost/carrental/admin” i adressfältet.
